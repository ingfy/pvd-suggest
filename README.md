# pvd-suggest: PrognosisVacationDateSuggestion
[![NPM version][npm-image]][npm-url]

The best module ever.


## InstallS

```bash
$ npm install --save ingfy/pvd-suggest
```


## Usage

```javascript
var pvdSuggest = require('pvd-suggest');
pvdSuggest('01'); // ['01.01.2011', ...]
```
